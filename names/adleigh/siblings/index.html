How we calculate sibling harmony

The harmony score is a weighted sum of five factors, each scored 0–100, then combined with fixed weights. Shared origin (30%) reflects whether two names come from the same cultural or linguistic background; same origin scores 100, partial 50–60, different 0. Phonetic rhythm (25%) rewards similar syllable count and optional first-letter match so names sound good when said together. Popularity band (20%) compares usage tier (top 100, top 500, top 1000, or other); names in the same band score higher so sibling names feel balanced in familiarity. Length balance (15%) and style cluster (10%) round out the formula. The result is deterministic: the same two names always get the same score. For Adleigh, the table above shows scores from 85–85.
